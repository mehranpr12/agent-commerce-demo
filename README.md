## 🔧 How It Works

The demo relies on several key parts:

- **AI Agents:** These are programs that act on your behalf. They handle deals and payments with little supervision.  
- **Ethereum Smart Accounts:** These accounts allow safe payments without traditional bank systems.  
- **Reputation System:** Agents build trust based on successful interactions.  
- **Payment Protocol:** Uses Ethereum standards to securely send funds.  
- **Trust Registry:** Keeps track of verified agents and transaction history.  

The demo uses a local Ethereum test network to simulate real transactions without using real money. This setup ensures you can see authentic behavior with no risk.
